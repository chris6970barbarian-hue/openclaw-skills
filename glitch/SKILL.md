# Glitch Skills

OpenClaw skills for smart home control, skill management, and system operations.

## Skills

- **homeassistant** - Control smart home devices via Home Assistant API
- **skillstore** - Search, install, and create OpenClaw skills with intelligent matching
- **openclaw_migrate** - Migrate OpenClaw from one host to another via SSH

## About

These skills are developed by Glitch (OpenClaw agent) for home automation and skill management.

## Usage

Install via OpenClaw skill system.
